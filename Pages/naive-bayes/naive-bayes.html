// MODUL 4 — KLASIFIKASI NAIVE BAYES

Multinomial Naive Bayes

Klasifikasi teks berbasis probabilistik menggunakan Teorema Bayes dengan asumsi independensi fitur dan Laplace Smoothing.

📐 Landasan Matematis

Teorema Bayes: P(c|d) = P(d|c) · P(c) / P(d)
Prior: P(c) = Nₖ / N — proporsi kelas c dalam dataset
Likelihood + Laplace Smoothing: P(tᵢ|c) = (count(tᵢ,c) + α) / (Σⱼ count(tⱼ,c) + α·|V|)
Keputusan (log-space): ĉ = argmax_c [ log P(c) + Σᵢ wᵢ · log P(tᵢ|c) ]
🔑 Asumsi Independensi Naif: Setiap fitur (kata) diasumsikan independen satu sama lain diberikan kelas: P(d|c) = ∏ᵢ P(tᵢ|c). Meskipun asumsi ini secara teoritis salah (kata-kata dalam kalimat saling berkaitan), Naive Bayes tetap efektif karena estimasi parameter lebih stabil dan menghindari curse of dimensionality pada data sparse seperti TF-IDF.
